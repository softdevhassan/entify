User Guide

Everything you need to know about using our comparative analysis platform.

1 Inputting Your Text

Start by pasting the content you want to analyze into the main text area on the Tool page. For the best result, we recommend using sentences that mention names, organizations, or locations.

You can use the samples provided at the bottom of the input area to see how the system handles different types of news or business text.

2 Choosing an Execution Mode

Our platform lets you choose exactly how you want to run the analysis. You can look at the CRF model to see how statistical weights work, or switch to the Neural mode to see the power of spaCy.

If you want to see them go head-to-head, use the Comparison mode. This will run both models simultaneously and show you a detailed breakdown of their performance differences.

3 Understanding the Results

Once the analysis is complete, the results will appear below the input section. We've designed this to be very visual so you can quickly see the differences.

  • Entity Highlights

    The original text will be color-coded based on the type of entity found, such as blue for people or green for locations.

  • Performance Metrics

    You'll see exactly how long each model took to process the text, measured in milliseconds, and the confidence level of the findings.
